PaiChai App H 프로젝트

## 기술스택
- Android Studio (Kotlin or Java)
- Spring Boot (Backend)
- MySQL (DBeaver 사용)
- Postman (API 테스트)

## 팀원 역할
- A: 안드로이드 앱 개발
- B: 백엔드 API 개발
- C: 데이터베이스 설계
- D: API 테스트 및 문서화

## 브랜치 전략
- `main`: 배포용 최종 버전
- `dev`: 전체 개발 브랜치
- `feat/*`: 기능 개발용
- `fix/*`: 버그 수정용

##  커밋 메시지 규칙
- `feat`: 새로운 기능 추가
- `fix`: 버그 수정
- `docs`: 문서 추가/수정
- `chore`: 설정, 폴더 구조 등
- `refactor`: 리팩토링
